# Nodejs-assignment-

FrontEnd : React.js
BackEnd : Node.js
DataBase : MongoDB

follow step for run front end project:
-> npm i 
-> npm run start 

follow step for run backend project:
->npm i
->node server.js

Api List for get:
-> / : All Books Data

Api List for Post :
-> /create-books : Create new books
-> /edit-book : Edit Book details
-> /view-book : View Book details
-> /delete-book : Delete Book details
